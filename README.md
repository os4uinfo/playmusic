# Playmusic
play douban.fm music on cli.

### Install
Ubuntu
```bash
$ sudo apt install libnotify-bin
$ sudo apt install mplayer
$ cd $GOPATH/src/
$ git clone https://github.com/os4uinfo/playmusic.git
$ cd playmusic
$ go build 
```
### Listen Music in Terminal
```bash
$ cd $GOPATH/src/playmusic
$ ./playmusic
```
